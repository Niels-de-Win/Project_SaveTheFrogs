=======================  FROG CROSSER V1.0  =======================

# What does it do?
Frog crosser is an iot-based project designed to give frogs easier crossing of roads. It works by continuously taking pictures of a certain area of land, followed by using AI to scan the pictures on the presence of frogs.
If no frogs are present, a light will shine red, indicating that there is no frog. If there is clearly a frog, the light turns green, indicating a frog.
Besides shining a light, the program also saves any pictures with frogs present, allowing for the viewing of previous captures.

# Why the project?
We've created frog crosser to lessen the amount of frogs dying while trying to cross the road every day.
